## 题干整理后可以得出以下信息

求：如何营救爷爷成功率最高  
推导：在给定营救模式下计算求成功率，找出成功率最高的营救模式  
推导：生成多重营救模式  


单人营救成功率=1/7  
双人营救成功率=1/7*2  
营救天数=左轮弹夹数=6  
```
# 暂不执行
营救模式为：每次n人营救，共营救6/n或6/n+1次，两种模式。后者包括一次人数不足n的强制救援  
```

规定游戏流程：
1. 产生葫芦娃
2. 选择是否营救
   1. 营救
      1. 判断是否成功
3. 进行轮盘赌
   1. 最后一天
      1. 死亡
   2. 死亡
      1. 游戏结束
   3. 存活
      1. 游戏继续
